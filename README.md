# rta-paper-data

Simulation inputs (structures, topologies, and parameters) to reproduce the molecular dynamics simulations reported in:

Rinto Thomas, Praveen Ranganath Prabhakar, and Michael von Domaros,
**A Residence-Time Approach for Determining Position-Dependent Diffusivities from Biased Molecular Simulations**, arXiv:submit/7435766 (2026).
https://arxiv.org/submit/7435766

## Overview

This repository provides ready-to-run input decks for molecular dynamics simulations and analysis workflows associated with the residence-time approach for determining position-dependent diffusivities from biased simulations.

Each system directory is self-contained and includes the system coordinates, topology, parameters, and simulation configuration files used in the study.
